### Development
---------------
Create Virtual Enviroment 
```
Python3 -m venv venv
```
Activate It
```
source venv/bin/activate
```
Install Flask
```
pip install flask
```
Set Flask ENV Variables
```
export FLASK_APP=app
export FLASK_ENV=development
```
Run Flask
```
flask run
````
If you need to change the port
```
flask run -p 5001
```
